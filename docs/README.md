#구현 할 기능 목록
- 도메인
  - 플레이어
    - 플레이어 이름
    - 플레이어 3자리의 수
  - 게임 매니저
    - 총 게임 회수
    - 게임 진행 중 여부
    - 시도 회수
    - 상대방
    - 도전자
- 서비스 로직
  - 매니저 초기화(싱글톤)
  - 게임 세팅
    - 게임 진행 여부 체크
    - 총 게임 회수 증가
    - 시도 회수 초기화
    - 상대방 / 도전자 생성
    - 상대방 플레이어 3자리의 수 임의의 수 초기화
  - 게임
    - 숫자 비교
    - 플레이어 입력
    - 결과 출력
    - 게임 진행 선택
    - 예외 처리
- 컨트롤러
  - 서비스 호출
- 화면
  - 컨트롤러 호출
- 단위 테스트
